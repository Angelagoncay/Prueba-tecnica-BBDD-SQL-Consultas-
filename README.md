**PROYECTO** Evaluación Final Módulo 2.

**Prueba técnica** Realización de los códigos de 24 consultas de SQL sobre la BBDD de Sakila.

**Contenido** Sakila es una Base de Datos de ejemplo que simula una tienda de alquiler de películas. 
Contiene información como películas, actores, clientes, alquileres, categorías, entre otras, e información(atributos) en columnas basadas en datos relevantes de cada una de las tablas. 
El ejercicio consiste en poner en práctica desde las queries (consultas) básicas, visualizando la información con SELECT y FROM , filtrada la misma con WHERE, LIKE , 
explorando datos ordenadamente con ORDER BY , manejando datos “entre varios” BETWEEN , incluso usando alias AS para renombrar columnas y hacer más sencilla la visualización de los datos. 
Utilizar funciones para consultar promedios y contar, cuentos como VG y COUNT. Combine datos de varias tablas para obtener una vista completa y útil con la utilización de JOIN´s. 
Organizar nuestros datos con GROUP BY, HAVING . Y realizar subconsultas, entre otros. 
Con el fin de poner en práctica todo lo aprendido y mostrar la destreza obtenida en el entorno SQL.
